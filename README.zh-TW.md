# 元數據去除劑

此應用程序是一種命令行工具，用於刪除所選目錄中的圖像元數據。該應用程序允許用戶選擇用於刪除或刪除的元數據配置文件。輸出圖像的質量也可以調整。處理每個文件後，應用程序將新圖像寫入沒有元數據的新圖像將圖像的原始目錄中的子目錄“無元數據”寫入。該應用程序具有簡單的用戶界面，並使用ImageMagick庫來處理圖像。
