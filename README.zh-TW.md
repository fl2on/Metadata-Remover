# 元資料刪除器

該應用程式是一個命令列工具，用於刪除選定目錄中的影像元資料。該應用程式允許用戶選擇要刪除哪些元資料設定檔或全部刪除。輸出影像的品質也可以調整。處理每個檔案後，應用程式將不帶元資料的新影像寫入原始影像目錄中的子目錄「WithoutMetadata」。該應用程式具有簡單的用戶介面，並使用 ImageMagick 庫來處理圖像。
